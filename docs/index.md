# INFRA // REFERENCE

A working reference covering the architecture underneath large enterprise environments,
built while preparing for VP-level IC engineering roles in banking. Each layer goes deep
enough to use as both interview prep and a long-term technical reference.

!!! note "How this is organized"
    Each layer below is broken into sub-layers to keep any single page from becoming
    unmanageable. Every concept page follows the same structure: definition, internal
    mechanism, why it exists, enterprise/banking reality, operational considerations,
    common misconceptions, and an interview angle.
